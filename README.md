# soopy dodge

a fun arcade style dodge game built with python and pygame.

control snoopy and dodge hearts flying across the screen.  
the game gets faster over time, making survival harder! 

# how to play

- ↑ → Move Up
- ↓ → Move Down

avoid the hearts for as long as possible, silly little game with a very real reminder<3

# tech stack

- python
- pygame

# run the game
install pygame:

```bash
pip install pygame
```

Run the game:

```bash
python main.py
```
